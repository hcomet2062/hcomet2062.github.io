I came up with this app this summmer while playing Twister with my siblings We wanted to be as random as possible so we decided to randomly generate each move.

Based on the few top searches, some drawbacks were that some moves would be redundant or that previous moves were not displayed. That in itself was problematic since we sometimes forgot what space we should be on.

For improvements, I would try to come up with a more aesthetically pleasing app. The app is fairly dull with its default settings, so that could be something to work on.
